# 📊 Techno Sales Power BI Dashboard

## 📌 Project Overview

This project presents an interactive Power BI dashboard developed using the Techno Sales dataset during my Hunar Intern internship. The dashboard provides business insights through interactive visualizations and DAX measures.

---

## 🚀 Dashboard Features

- Interactive Order Status Slicer
- KPI Cards (Total Sales, Best Category, Top Brand)
- Category-wise Sales Analysis
- Brand-wise Sales Distribution
- State-wise Sales Distribution Map
- Key Insights Panel
- Professional 16:9 Dashboard Design

---

## 🛠 Tools Used

- Microsoft Power BI Desktop
- Power Query
- DAX (Data Analysis Expressions)
